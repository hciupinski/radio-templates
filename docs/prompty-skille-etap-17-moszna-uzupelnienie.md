## Etap 17: USG moszny - uzupełnienie zakresu

### 1. `radiology-template-localizer`

```text
Przygotuj polskie szablony USG moszny dla skrętu przydatka jądra, zapalenia jądra, torbieli najądrza lub spermatocele, mikrolitiazy jądra i przepukliny pachwinowo-mosznowej.
W mikrolitiazie podaj zalecenia kontroli tylko zgodne z czynnikami ryzyka ESUR. Przy przepuklinie opisz odprowadzalność i cechy uwięźnięcia. Użyj wyłącznie konkretnych źródeł.
```

### 2. `radiology-atlas-yaml-writer`

```text
Zapisz pięć szablonów YAML w odpowiednich katalogach USG moszny, rozszerz taxonomy.yaml i sources.yaml, pomiń imageRefs bez pewnych bezpośrednich URL oraz uruchom npm run validate.
```
